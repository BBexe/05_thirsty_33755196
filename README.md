# The Thirsty Student Shop

A web application for an online shop built with Node.js and Express.

## Description

This project is a simple  website that includes various features like product browsing, search functionality, customer surveys, user registration, and an interactive game called "No-e in Market".

## Features

- **Home Page**: Displays available product categories
- **About Page**: Shows information about shop locations and managers
- **Search**: Search for products by name and category
- **Survey**: Customer feedback form with various question types
- **Registration**: User registration system
- **No-e Game**: Interactive game where users identify items without the letter 'e'

## Installation

1. Clone this repository
2. Install dependencies:
   ```
   npm install
   ```

## Running the Application

Start the server:
```
node index.js
```

The application will run on `http://localhost:8000`

## Technologies Used

- Node.js
- Express.js
- EJS (Embedded JavaScript templating)
- CSS
